# Avaliador de Expressões Matemáticas

Este programa avalia expressões matemáticas em notação infixa ou pós-fixa, realizando conversões entre as notações e calculando seus valores.

## Funcionalidades

- Conversão entre notações infixa e pós-fixa
- Cálculo do valor de expressões matemáticas
- Suporte para operadores básicos (+, -, *, /, %, ^)
- Suporte para funções matemáticas (sen, cos, tan, log, raiz)
- Tratamento robusto de erros e validação de sintaxe

## Compilação e Execução

Para compilar o programa, execute:

```bash
gcc expressao.c main.c -o avaliador -lm
